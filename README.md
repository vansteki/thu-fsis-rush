thu-fsis-rush
=============
## THU 選課大作戰!
## Require
  Mozilla Firefox >= 18.0
  
  For Windows only
  
  Test on Windows7, Ruby1.8.7, Firefox19
## Usage
    fsis.exe 
    >"Enter ID"
    輸入學號
    >"Enter Password"
    輸入密碼 (輸入過程不會被看到)
    "Rush Times?"
    輸入重試次數
## Dev
