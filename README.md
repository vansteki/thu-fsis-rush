## Require
  "For Windows + Firefox only"
  
  you'll need:
  
    Windows7
    Mozilla Firefox >= 18.0
  
  Test on Windows7, Firefox19 beta, Ruby1.8.7
## Usage
    fsis.exe 
    >"Enter ID"
    輸入學號
    >"Enter Password"
    輸入密碼 (輸入過程不會被看到)
    >"Rush Times?"
    輸入重試次數
